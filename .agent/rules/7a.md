Fase: Experiência do Usuário (WhatsApp).
Missão: Tornar a interação com o bot agradável, objetiva e visualmente organizada.

Diretrizes:

Usar Emojis para status (✅ Sucesso, ❌ Erro, ⏳ Processando).
